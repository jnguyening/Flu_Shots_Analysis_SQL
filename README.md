# Healthcare Immunization Compliance Analysis

## Overview

Identifying gaps in immunization is a critical task for healthcare providers seeking to improve community health outcomes and ensure equitable care. This project utilizes PostgreSQL to analyze a clinical dataset, defining a specific "eligible cohort" based on age, mortality status, and recent medical encounters. By joining these demographics with immunization logs, the analysis quantifies the 2022 flu shot compliance rate and highlights disparities across various population segments.

The technical implementation features advanced SQL techniques like Epoch-based age calculation and Case-driven feature engineering. These insights allow healthcare stakeholders to target specific counties or demographic groups for localized health interventions.
